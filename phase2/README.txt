Setup Instructions:
1. Install Android Studio module in IntelliJ or Android Studio IDE
2. VCS -> Get from version control, paste link to repository, if prompted to build project click no
Or if you're on the start screen, click on Get from version control, paste link to repository, if prompted to build project click no
3. File -> New -> Project from existing sources -> choose phase2 as root
Or if you're on the start screen click on import project -> choose phase2 as root
4. Click import gradle project from pop-up on lower right corner and wait for about a minute
5. Open AVD Manager and Create Virtual Device phone with API 26 (Pixel 3 Oreo)


Notes:
View Alerts, Memos and Events button outputs to System.out.
